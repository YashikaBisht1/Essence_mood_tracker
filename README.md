## 🌟 Overview

Essence is an immersive mood tracking and emotional exploration application built with Next.js. It provides a unique, interactive way to track, understand, and engage with your emotions through visual representations and AI-powered interactions.

## ✨ Features

- **Mood Bottle**: Visualize your current mood with beautiful, color-coded bottles
- **Mood Rituals**: Guided emotional exercises and reflections
- **Mood Collection**: Track and review your emotional patterns over time
- **Luna Companion**: AI-powered emotional support and conversation
- **Alter Ego Simulator**: Explore different aspects of your personality
- **Mood Blender**: Mix and match emotions to discover new emotional states

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ and npm/pnpm
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/YashikaBisht1/Essence_mood_tracker.git
   cd Essence_mood_tracker
   ```

2. Install dependencies:
   ```bash
   pnpm install
   # or
   npm install
   ```

3. Run the development server:
   ```bash
   pnpm dev
   # or
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

## 🛠️ Tech Stack

- **Frontend**: Next.js 13+ with TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI Primitives
- **Icons**: Lucide React
- **State Management**: React Hooks
- **Form Handling**: React Hook Form
- **Date Handling**: date-fns
